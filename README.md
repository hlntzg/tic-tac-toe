### Features
- Turn-based gameplay (X and O)
- Dynamic board rendering with React
- Win detection logic
- Responsive layout

### Future features
- Game restart button
- Winner highlight
- Move history and time travel
- Better styling

### Getting started
```bash
npm install
npm start
```
